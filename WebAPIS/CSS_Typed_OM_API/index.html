CSS Typed Object Model (Typed OM) API

The CSS Typed OM API provides developers with a powerful, modern way to programmatically interact with CSS properties. Instead of dealing with raw strings, it allows manipulation of styles using structured, type-safe objects.

Key Features

  • Typed Values: Work with specific CSS value types like CSSUnitValue and CSSKeywordValue.
  • Improved Performance: Avoid frequent string parsing, ensuring faster and safer style manipulations.
  • Mathematical Operations: Perform calculations on CSS values using CSSMathValue.
  • Seamless Integration: Leverage the API for animations and dynamic styling.

Benefits

  • Type-safe manipulation of CSS styles.
  • Improved code readability and maintainability.
  • Enhanced performance for complex style calculations.
  • Works well with modern JavaScript frameworks and libraries.

Methods and Properties

  • element.computedStyleMap(): Retrieves a map of the computed styles for an element.
  • element.attributeStyleMap.set(property, value): Sets a style property using typed values.
  • CSS.px(value): Creates a pixel unit value.
  • CSS.keyword(value): Creates a keyword value (e.g., 'auto', 'none').
  • CSSMath.add(...values): Performs addition on CSS values.
