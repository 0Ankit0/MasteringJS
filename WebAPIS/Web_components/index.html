Web Components Guide

Learn how to create reusable UI components with Web Components.

ðŸ”¹ What are Web Components?

Web Components is a set of standards that allow developers to create custom, reusable HTML elements. These custom elements can encapsulate functionality and style, providing a way to create modular, maintainable components that can be used across different projects.

Web Components consist of four main parts:

  • Custom Elements: Define new HTML elements with their own behavior.
  • Shadow DOM: Encapsulate the internal structure of the element.
  • HTML Templates: Define the markup structure of the element that can be reused.
  • HTML Imports (deprecated but still relevant in some contexts): Import and reuse components.

ðŸ”¹ How Web Components Work

Web Components allow us to build custom elements that are reusable and independent of other frameworks. Here are the steps to create a Web Component:

  • Define a class that extends the base class HTMLElement.
  • Define the template that contains the HTML structure and the shadow DOM.
  • Define the lifecycle methods like connectedCallback() for adding functionality.
  • Register the custom element using customElements.define().

ðŸ”¹ Example: Simple Web Component

Below is an example of how to create a basic Web Component:


// Defining a new custom element
class MyButton extends HTMLElement {
  constructor() {
    super();
    // Attach shadow DOM
    const shadow = this.attachShadow({ mode: 'open' });

    // Create and style the button
    const button = document.createElement('button');
    button.textContent = 'Click me!';
    button.style.backgroundColor = 'skyblue';
    button.style.fontSize = '16px';
    button.style.padding = '10px 20px';

    // Append the button to shadow DOM
    shadow.appendChild(button);

    // Add event listener
    button.addEventListener('click', () => {
      alert('Button clicked!');
    });
  }
}

// Register the custom element
customElements.define('my-button', MyButton);

Explanation:

  • We define a new class, MyButton, extending HTMLElement.
  • Inside the constructor, we attach a shadow DOM and create a button element.
  • We style the button and append it to the shadow DOM. We also add an event listener for the click event.
  • Finally, we register the new element using customElements.define(), making it available to use in HTML as <my-button>.
