What is the DOM?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, making it possible to manipulate the document dynamically with programming languages like JavaScript.

Structure of the DOM

The DOM organizes the document into a tree structure, where each element, attribute, and text becomes a node. Here’s an example of how an HTML document translates into a DOM tree:

HTML Code

<html>
    <head>
        <title>Sample Page</title>
    </head>
    <body>
        <h1>Hello, World!</h1>
    </body>
</html>

DOM Tree Representation

  • html
    • head
      • title: "Sample Page"
    • body
      • h1: "Hello, World!"

Key Components of the DOM

  • Document: The root node representing the entire document.
  • Element: HTML tags like <div>, <p>, etc.
  • Attribute: Additional information about elements (e.g., class, id).
  • Text: Content inside an element (e.g., "Hello, World!").

How to Use the DOM

With JavaScript, you can manipulate the DOM to dynamically update the content of a webpage. Below is an example:

JavaScript Example

HTML:

<body>
    <h1 id="title">Old Title</h1>
    <button id="changeTitle" class="btn btn-primary">Change Title</button>
</body>

JavaScript:

document.getElementById("changeTitle").addEventListener("click", function() {
    const title = document.getElementById("title");
    title.textContent = "New Title";
});

Benefits of Using the DOM

  • Enables dynamic updates to web pages without reloading.
  • Allows interaction with HTML elements programmatically.
  • Works seamlessly with event listeners for user interaction.
