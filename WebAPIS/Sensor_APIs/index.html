Sensor API Documentation

This page explains how to use the Sensor API to detect various device sensors like Ambient Light, Accelerometer, Gyroscope, Magnetometer, and Orientation Sensors.

Introduction

The Sensor API provides access to various sensors on the device, enabling web applications to interact with and gather information from sensors like the Ambient Light Sensor, Accelerometer, Gyroscope, and others. These sensors can be used for various purposes such as motion detection, environment lighting analysis, and device orientation tracking.

1. Ambient Light Sensor

The Ambient Light Sensor detects the intensity of light in the environment.

Methods:

  • start(): Starts the sensor to begin collecting light data.
  • stop(): Stops the sensor from collecting light data.

Properties:

  • illuminance: The current light level in lux.
  • onreading: Triggered when a new reading is available.
  • onerror: Triggered if an error occurs.

Events:

  • reading: Event fired when the sensor provides updated data.
  • error: Event fired when an error occurs with the sensor.

2. Accelerometer

The Accelerometer sensor provides the acceleration of the device along the three axes (x, y, and z).

Methods:

  • start(): Starts the accelerometer to monitor motion data.
  • stop(): Stops collecting accelerometer data.

Properties:

  • x: Acceleration along the x-axis.
  • y: Acceleration along the y-axis.
  • z: Acceleration along the z-axis.
  • onreading: Triggered when new accelerometer data is available.
  • onerror: Triggered if an error occurs.

Events:

  • reading: Event fired when the accelerometer provides updated data.
  • error: Event fired when an error occurs with the accelerometer.

3. Gyroscope

The Gyroscope sensor measures the device's rotational velocity around the three axes (x, y, and z).

Methods:

  • start(): Starts the gyroscope to begin collecting rotation data.
  • stop(): Stops collecting gyroscope data.

Properties:

  • x: The rotational velocity around the x-axis.
  • y: The rotational velocity around the y-axis.
  • z: The rotational velocity around the z-axis.
  • onreading: Triggered when new gyroscope data is available.
  • onerror: Triggered if an error occurs.

Events:

  • reading: Event fired when the gyroscope provides updated data.
  • error: Event fired when an error occurs with the gyroscope.

4. Magnetometer

The Magnetometer sensor detects the Earth's magnetic field, helping to measure the device's orientation with respect to the Earth's magnetic poles.

Methods:

  • start(): Starts the magnetometer to begin collecting data.
  • stop(): Stops the magnetometer from collecting data.

Properties:

  • x: The magnetic field strength along the x-axis.
  • y: The magnetic field strength along the y-axis.
  • z: The magnetic field strength along the z-axis.
  • onreading: Triggered when new magnetometer data is available.
  • onerror: Triggered if an error occurs.

Events:

  • reading: Event fired when the magnetometer provides updated data.
  • error: Event fired when an error occurs with the magnetometer.

5. Orientation Sensor

The Orientation Sensor provides information about the device's orientation (alpha, beta, gamma), which helps detect how the device is rotated in space.

Methods:

  • start(): Starts the orientation sensor to begin monitoring the device's rotation.
  • stop(): Stops the orientation sensor.

Properties:

  • alpha: The device's rotation around the z-axis (0-360 degrees).
  • beta: The device's rotation around the x-axis (-180 to 180 degrees).
  • gamma: The device's rotation around the y-axis (-90 to 90 degrees).
  • onreading: Triggered when new orientation data is available.
  • onerror: Triggered if an error occurs.

Events:

  • reading: Event fired when the orientation sensor provides updated data.
  • error: Event fired when an error occurs with the orientation sensor.
