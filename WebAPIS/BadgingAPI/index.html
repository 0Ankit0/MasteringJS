Badging API

The Badging API allows web applications to set an application-wide badge, shown in an operating-system-specific place associated with the application (such as the shelf or home screen). This can be used to notify the user when the state of the application changes (e.g., when new messages have arrived).

Methods

  • set(unreadCount): Sets the badge with the specified number.
  • clear(): Clears the badge.

Implementation

Below is an example of how to use the Badging API:


// Check if the Badging API is supported
if ('setAppBadge' in navigator) {
  // Set the badge
  navigator.setAppBadge(5).catch((error) => {
    console.error('Failed to set badge:', error);
  });

  // Clear the badge
  navigator.clearAppBadge().catch((error) => {
    console.error('Failed to clear badge:', error);
  });
} else {
  console.log('Badging API is not supported in this browser.');
}
