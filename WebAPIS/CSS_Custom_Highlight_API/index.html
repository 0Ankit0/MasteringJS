CSS Custom Highlight API

The CSS Custom Highlight API allows developers to create custom text highlights that are independent of the built-in `` or selection-based highlights. These highlights can be styled using CSS and can be applied to any text content on the page.

Key Components

  • Highlight Registry: Manages highlight registrations
  • Range: Defines the text segment to highlight
  • HighlightRegistry.register(): Registers new highlights
  • ::highlight(): CSS pseudo-element for styling highlights

Methods & Properties

  • CSS.highlights: The global highlight registry
  • register(name, highlight): Registers a new highlight
  • delete(name): Removes a highlight registration
  • clear(): Removes all highlight registrations

Implementation Example


// Create ranges for the text to highlight
const range = new Range();
range.setStart(textNode, 5);
range.setEnd(textNode, 10);

// Create a highlight
const highlight = new Highlight(range);

// Register the highlight
CSS.highlights.set('my-highlight', highlight);

// Style the highlight in CSS
::highlight(my-highlight) {
  background-color: yellow;
  color: black;
}

Browser Support

The CSS Custom Highlight API is relatively new and currently supported in:

  • Chrome 105+
  • Edge 105+
  • Opera 91+
Note: This API is part of modern web platform features and may require feature detection before use in production environments.
