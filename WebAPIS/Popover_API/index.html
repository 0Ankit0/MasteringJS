HTML Native Popover API Guide

Learn how to use the new native popover attribute in HTML.

What is the Native Popover API?

The Native Popover API is a built-in HTML feature that allows elements to display small pop-up panels without requiring JavaScript.

HTML Attributes for Popover

The following attributes control popovers:

Attribute Description Example Usage
popover Defines the element as a popover <div popover>Content</div>
popover-target Specifies which popover should be triggered <button popover-target="popoverID">Click Me</button>
popover-auto Automatically closes the popover when clicking outside <div popover popover-auto>Auto Close</div>
hidden Hides the popover until triggered <div popover hidden>Hidden Popover</div>

CSS Features

Native popovers can be styled using CSS:


/* Style for all popovers */
[popover] {
    background: white;
    border: 1px solid #ccc;
    padding: 10px;
    border-radius: 5px;
    box-shadow: 0px 4px 8px rgba(0, 0, 0, 0.2);
    max-width: 300px;
}

/* Custom animation */
[popover]:popover-open {
    animation: fadeIn 0.3s ease-in-out;
}

@keyframes fadeIn {
    from { opacity: 0; transform: scale(0.9); }
    to { opacity: 1; transform: scale(1); }
}

JavaScript Methods

You can control popovers using JavaScript:


// Show the popover
document.getElementById("examplePopover").showPopover();

// Hide the popover
document.getElementById("examplePopover").hidePopover();

// Toggle the popover visibility
document.getElementById("examplePopover").togglePopover();

Events

Native popovers trigger the following events:

Event Description
beforetoggle Fires before the popover is shown or hidden
toggle Fires when the popover is shown or hidden

Example Usage of Events


document.getElementById("examplePopover").addEventListener("toggle", function(event) {
    console.log("Popover toggled!");
});
