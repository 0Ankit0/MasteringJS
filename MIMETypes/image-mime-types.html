Image MIME Types

Back to Overview

1. image/jpeg

What it represents: JPEG images.

Use case: Commonly used for digital photos and web images. 

Content-Type: image/jpeg

Example:

Content-Type: image/jpeg

2. image/png

What it represents: PNG images.

Use case: Used for images that require transparency and lossless compression. 

Content-Type: image/png

Example:

Content-Type: image/png

3. image/gif

What it represents: GIF images.

Use case: Used for simple animations and images with limited colors. 

Content-Type: image/gif

Example:

Content-Type: image/gif

4. image/webp

What it represents: WebP images.

Use case: Used for images with both lossy and lossless compression. 

Content-Type: image/webp

Example:

Content-Type: image/webp

5. image/svg+xml

What it represents: SVG (Scalable Vector Graphics) images.

Use case: Used for vector images that can be scaled without losing quality. 

Content-Type: image/svg+xml

Example:

<svg width="100" height="100" xmlns="http://www.w3.org/2000/svg">
          <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
        </svg>
