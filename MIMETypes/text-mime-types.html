Text-Based MIME Types

Back to Overview

1. text/plain

What it represents: Plain text files.

Use case: For representing unformatted text.

Content-Type: text/plain

Example:

Hello, world!

2. text/html

What it represents: HTML (HyperText Markup Language) files.

Use case: For creating web pages and web applications. 

Content-Type: text/html

Example:

<!DOCTYPE html>
<html>
<head>
  <title>Example</title>
</head>
<body>
  <h1>Hello, world!</h1>
</body>
</html>

3. text/css

What it represents: CSS (Cascading Style Sheets) files.

Use case: For defining the look and layout of HTML elements. 

Content-Type: text/css

Example:

body { color: blue; }

4. text/javascript

What it represents: JavaScript files.

Use case: For running client-side JavaScript in browsers. 

Content-Type: text/javascript

Example:

console.log("Hello, world!");

5. text/csv

What it represents: CSV (Comma-Separated Values) files.

Use case: For representing tabular data in a plain text format. 

Content-Type: text/csv

Example:

name,age
John,30
Jane,25

6. text/xml

What it represents: XML (eXtensible Markup Language) files.

Use case: For representing structured data in a text format. 

Content-Type: text/xml

Example:

<person>
  <name>John</name>
  <age>30</age>
</person>
