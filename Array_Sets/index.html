Understanding JavaScript Arrays and Sets

Learn about the built-in collection types in JavaScript, including their methods, parameters, and use cases.

Arrays in JavaScript

An array is an ordered collection of values. Arrays can store elements of any type (numbers, strings, objects, etc.) and come with a wide range of built-in methods.

Common Array Methods

  • push(item): Adds one or more items to the end of the array.
  • pop(): Removes and returns the last element of the array.
  • shift(): Removes and returns the first element.
  • unshift(item): Adds one or more elements to the beginning of the array.
  • splice(start, deleteCount, ...items): Removes or replaces existing elements and/or adds new ones.
    Example: splice(1, 2, 'a', 'b') starts at index 1, deletes 2 items, and inserts 'a' and 'b'.
  • slice(start, end): Returns a shallow copy of a portion of the array from start to end (not inclusive).
  • map(callback): Creates a new array with the results of calling a provided function on every element.
    Callback parameters: currentValue, index, array
  • filter(callback): Creates a new array with all elements that pass the test implemented by the provided function.
    Callback parameters: currentValue, index, array
  • reduce(callback, initialValue): Executes a reducer function on each element, resulting in a single output value.
    Callback parameters: accumulator, currentValue, index, array

Example: Array Manipulation


// Create an array
const fruits = ['apple', 'banana', 'cherry'];
console.log("Initial array:", fruits);

// Add an element at the end
fruits.push('date');
console.log("After push:", fruits);

// Remove the last element
const lastFruit = fruits.pop();
console.log("After pop (removed):", lastFruit, fruits);

// Add an element at the beginning
fruits.unshift('avocado');
console.log("After unshift:", fruits);

// Remove the first element
const firstFruit = fruits.shift();
console.log("After shift (removed):", firstFruit, fruits);

// Using splice to remove and add elements
// Remove 1 element at index 1 and add 'blueberry' and 'cantaloupe'
fruits.splice(1, 1, 'blueberry', 'cantaloupe');
console.log("After splice:", fruits);

// Using map to transform array elements
const upperFruits = fruits.map(fruit => fruit.toUpperCase());
console.log("Mapped array (upper case):", upperFruits);

// Using filter to select items that start with 'c'
const cFruits = fruits.filter(fruit => fruit.startsWith('c'));
console.log("Filtered array (starts with 'c'):", cFruits);

// Using reduce to concatenate all fruit names
const allFruits = fruits.reduce((acc, fruit) => acc + ', ' + fruit);
console.log("Reduced result:", allFruits);

Notice how each method modifies or creates a new array. The parameters for methods like splice or the callback functions for map and reduce are essential for controlling how the array is transformed.

Sets in JavaScript

A Set is a collection of unique values. Unlike arrays, sets do not store duplicate items and are especially useful when you need to enforce uniqueness.

Common Set Methods

  • add(value): Adds a new element to the set. If the element already exists, the set remains unchanged.
  • delete(value): Removes a specified element from the set; returns true if the element was removed.
  • has(value): Returns true if the set contains the specified element.
  • clear(): Removes all elements from the set.
  • size: A property that returns the number of elements in the set.

Example: Working with Sets


// Create a new set
const numbers = new Set();

// Add values to the set
numbers.add(1);
numbers.add(2);
numbers.add(3);
numbers.add(2); // duplicate, will not be added
console.log("Set after adding elements:", numbers);

// Check if a value exists
console.log("Set has 2:", numbers.has(2));
console.log("Set has 4:", numbers.has(4));

// Get the size of the set
console.log("Size of set:", numbers.size);

// Remove a value
numbers.delete(2);
console.log("Set after deleting 2:", numbers);

// Iterate over set values
numbers.forEach(num => console.log("Iterating set value:", num));

// Clear the set
numbers.clear();
console.log("Set after clear:", numbers);

Sets are particularly useful when you need to store unique items. For instance, if you want to remove duplicates from an array, you can create a set from that array: 


// Remove duplicates from an array using a Set
const duplicateArray = [1, 2, 2, 3, 3, 3, 4];
const uniqueArray = [...new Set(duplicateArray)];
console.log("Unique array:", uniqueArray);

Arrays vs. Sets: When to Use Which

Arrays are ordered collections that can contain duplicate items and are best used when the order of elements matters or when you need to perform operations like mapping or filtering on a list.

Sets enforce uniqueness and provide faster lookup times for checking if an item exists. Use sets when you need to ensure that all elements are unique and when order is not a priority.
